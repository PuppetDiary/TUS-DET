# TUS-DET
Open repo for paper: TUS-DET: Open-Vocabulary Pretraining with Standard Planes for Few-Shot Thyroid Ultrasound Lesion Detection

# Abstract
Ultrasound imaging plays a vital role in thyroid disease diagnosis; however, the performance of existing deep learning models remains limited in few-shot scenarios, primarily due to the high cost and expertise required for medical data annotation. To address this challenge, we propose TUS-DET (Thyroid Ultrasound Detection), a novel image-text multimodal object detection framework tailored for resource-constrained medical applications. TUS-DET leverages thyroid ultrasound standard planes and key anatomical structure descriptions to introduce an open-vocabulary pre-training strategy, enhancing model generalizability and lesion localization under limited supervision. Experimental results on four public thyroid ultrasound datasets demonstrate that TUS-DET consistently outperforms representative baselines in few-shot detection and classification tasks. For example, with only 30 annotated samples, our model achieves higher detection accuracy than existing methods, and TUSP-based pre-training substantially improves performance on lesion detection and classification, especially in low-data regimes. Furthermore, TUS-DET maintains moderate computational demands, making it suitable for practical clinical deployment. To facilitate future research, we will release the code, datasets with bounding box annotations, and pre-trained models upon acceptance of the paper.
